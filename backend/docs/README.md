# Documentation Index

This folder contains the current backend-facing documentation set for the repository as implemented in code at the time of review.

Files:

- `codebase-analysis.md`: deep architectural analysis of the full codebase, including backend, frontend integration, data model, and workflow behavior.
- `backend-api-documentation.md`: comprehensive backend API reference covering auth, dashboards, mutations, errors, uploads, and socket events.
- `backend-api-critique.md`: professional critique of the API and supporting backend design, with concrete risks and maturity gaps.
- `backend-api-improvement-todo.md`: prioritized improvement backlog to move the API toward professional production standards.

Scope:

- Backend code reviewed from `backend/`
- Database bootstrap reviewed from `database/mbciansappointizen.sql`
- Frontend integration behavior reviewed from `frontend/shared/*` and role-specific pages

Important note:

- There is an older root-level file named `BACKEND_API_DOCUMENTATION.md`. The documents in this folder are intended to be the reviewed, implementation-aligned set.
